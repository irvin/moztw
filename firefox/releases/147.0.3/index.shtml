Firefox 新鮮事

  • 全新功能

    改進 CSS Anchor Positioning 與 Navigation Web API 的互通性表現。

  • 修正

    修正部分網站在特定 :hover 互動後，滑鼠滾輪捲動時 position: sticky 元素可能卡住或不更新的回歸問題。（Bug 2010481）

  • 修正

    修正包含跨來源 iframe 的頁面，在使用 Inspector 節點挑選器並重新整理後，開發者工具可能無法再次開啟的問題。（Bug 2003810）

  • 修正

    修正 DNS over HTTPS 供應商設定區塊偶爾顯示為空白，導致無法查看或調整目前設定的問題。（Bug 2010501）

  • 修正

    修正 Windows 安裝大量字型時，Firefox 介面（分頁、選單、設定）可能顯示亂碼或符號而非可讀文字的問題。（Bug 2012950）

  • 修正

    若需瞭解此版本之前的變更，請參考 Firefox 147.0.2 版本資訊。

已知問題

  • 尚待修復

    在 Linux（Wayland）系統上，部分彈出視窗與情境選單仍可能出現在錯誤位置（例如視窗左上角）。Firefox 147.0.3 已先提供部分緩解，剩餘案例將在後續版本修正。（Bug 2003045）
