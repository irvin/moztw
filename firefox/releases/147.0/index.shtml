Firefox 新鮮事

  • 全新功能

    現在可在 Firefox 中讓 PiP 視窗隨時自動開啟。只要到設定啟用「自動開啟 Picture-in-Picture」，當你切換分頁時，正在播放的影片就會自動跳出成 PiP 視窗，追劇或看教學更不中斷。

    此功能逐步釋出中：某些新的 Firefox 功能採行逐步釋出（progressive roll out），意味著某些使用者會比其他人更早看到此功能。如此有助於獲取早期回饋，迅速發現錯誤與改善行為，從而提升整體 Firefox 使用者的體驗。

  • 修正

    修正並改善垂直分頁模式下的拖放行為，讓分頁操作更穩定、直覺。

  • 修正

    修正 Windows 上 Firefox 在次螢幕最大化時，點擊螢幕頂端分頁偶爾失效的問題。

  • 修正

    修正因情境選單項目過長，導致「全部分頁列表」開啟延遲的效能問題。

  • 修正

    修正 Android 上某些觸控事件會被錯誤處理為滑鼠事件的問題。

  • 修正

    修正 13 代 Intel 與 7000 系列 AMD CPU 裝置上，部分進階使用者回報的偶發當機問題。

  • 修正

    修正 Linux 上匯出密碼 CSV 時，檔案名稱中的括號會被編碼成 URL 字元的問題。

  • 修正

    修正讓某些使用者在 YouTube 觀看影片時出現畫面凍結的問題。

  • 修正

    修正進入睡眠後喚醒，部分 VPN 擴充套件的連線可能失效的問題。

  • 修正

    多項安全性修正。

  • 企業版

    你可以在 Firefox 147 企業版版本資訊 中找到政策更新與企業版相關的錯誤修復資訊。

  • 開發功能

    Firefox 147 開發者資訊

  • 開發功能

    新增 JavaScript Error Report API，讓網站可在開發者工具主控台中更精準呈現錯誤。

  • 開發功能

    新增主控台的 inline 變數預覽，可直接在程式碼行內查看當下變數值。

  • 網路平台

    Firefox 現在支援 URLPattern API，可用於網址比對與路由處理。

  • 網路平台

    支援 CSS 偽類 :open，可在 <details> 等元件展開時套用樣式。

  • 網路平台

    新增 Web API StorageAccessHandle.revokeObjectURL()，可在儲存存取情境中撤銷物件 URL。

  • 網路平台

    現在可在 anchor element 上設定 Preference API。

  • 網路平台

    Windows 使用者的媒體回放速度（playback rates）現在可跨不同網站保留設定。

  • 網路平台

    新增 CSS 屬性 scrollbar-gutter 與 shape-margin 的支援。

  • 網路平台

    新增 scrollbar-width 與 scrollbar-color 在 Android 上的支援。

  • 網路平台

    新增 API SpeechSynthesis.onvoiceschanged 與 SpeechSynthesis.getVoices() 的支援。

  • 網路平台

    修正 DevTools 並改進 Firefox 內建截圖工具的穩定性與使用體驗。

  • 請參考此版本的完整變更清單。 你可能也想瞭解 版本的更新。
