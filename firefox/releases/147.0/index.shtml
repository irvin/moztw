Firefox 新鮮事

  • 全新功能

    自動填寫地址（Address autofill）已對瑞士使用者開啟。

  • 全新功能

    使用 Apple Silicon 處理器的 macOS（支援版本）現在也全面啟用 WebGPU 支援。

  • 全新功能

    在具備 AMD GPU 的系統上啟用硬體解碼影片的 zero-copy 播放，讓影片播放效能與 Intel、NVIDIA GPU 相當。

  • 全新功能

    Firefox 現在支援 Safe Browsing V5 通訊協定，並逐步從 V4 轉為 Safe Browsing V5 協定的 本機威脅清單模式。

  • 全新功能

    將加強型追蹤保護（ETP）設定為「嚴格」的使用者，將預設啟用區域網路存取限制；瀏覽器會要求使用者明確允許網站存取區域網路資源。

    此功能逐步釋出中：某些新的 Firefox 功能採行逐步釋出（progressive roll out），意味著某些使用者會比其他人更早看到此功能。如此有助於獲取早期回饋，迅速發現錯誤與改善行為，從而提升整體 Firefox 使用者的體驗。

  • 全新功能

    現在可在 about:keyboard 啟用測試自訂快捷鍵功能，取代那些難打或難記的按鍵組合，或避免與其他軟體衝突。歡迎到 Mozilla Connect 分享回饋。

  • 全新功能

    Firefox 現已支援 Freedesktop.org 的 XDG Base Directory Specification。

  • 全新功能

    現在分頁切換到背景時，可自動以子母畫面顯示該分頁上播放中的影片。特別感謝我們的志工 Daniele Ferla 貢獻此功能！

  • 修正

    修正部分 Windows 使用者在 Firefox 視窗最大化且游標在螢幕頂端時，無法選到分頁的問題。

  • 修正

    修正 HTTP/3 請求若含非 UTF-8 檔頭值時，可能逾時或退回 HTTP/2 的問題。

  • 修正

    可拖曳的按鈕現在可直接從按鈕本身開始拖曳。

  • 修正

    Linux GNOME Mutter 使用者的視窗與繪圖表面尺寸，已更新為符合實際像素格點，因此任何視窗尺寸下，Firefox 在非整數縮放螢幕上渲染都會更清晰。

  • 修正

    多項安全性修正。

  • 變更

    Firefox 現在採用與其他主要瀏覽器相同的 Accept-Language 品質數值（q-values），第二語言偏好從原本 q=0.5 調整為 q=0.9，之後每項遞減 0.1（最低 0.1）；可修復部分伺服器不正確的拒絕較低品質值的相容性問題。

  • 企業版

    你可以在 Firefox 147 企業版版本資訊 中找到政策更新與企業版相關的錯誤修復資訊。

  • 開發功能

    Firefox 147 開發者資訊

  • 開發功能

    View Transitions：transition 的虛擬元素現在會出現在元件面板中，相關動畫也可在動畫面板查看。

  • 開發功能

    Anchor positioning：具有有效 anchor-name 的元素在 Elements 視圖會顯示 anchor 徽章；當選取使用 position-try-fallbacks 的元素時，@position-try 規則會顯示在 CSS 規則面板。

  • 開發功能

    JSON 檢視器加入匯入到 Firefox Profiler 的按鈕，方便分析資源大小。

  • 開發功能

    現在可在 CSS 規則面板新增／編輯 pseudo-element selectors。

  • 網路平台

    支援 Navigation API，此 API 可處理瀏覽器瀏覽時的啟始、攔截與管理行為。 這是繼先前的網路平台功能，如 History API 與 window.location 的後繼者，解決了它們的短板。

  • 網路平台

    更新 Unicode ICU 函式庫至 release 78，加入 Unicode 17 與更多語系支援。

  • 網路平台

    支援 service workers 中的 ES module，跟進其他主要瀏覽器，改善現代 Web App 相容性。

  • 網路平台

    新增 CSS Module Scripts，可用 JavaScript 模組系統匯入樣式表。

  • 網路平台

    現在可在 ::marker 虛擬元素上使用 CSS 的 counter-* 與 quotes 屬性。

  • 網路平台

    CompressionStream 與 DecompressionStream 現支援 Brotli 格式。

  • 網路平台

    支援 :active-view-transition-type 選擇器，並支援相關 View Transitions API 的變更。

  • 網路平台

    Firefox 現在透過 Document.activeViewTransition 揭露目前文件作用中的 view transition 資訊。

  • 網路平台

    新增 CSS anchor positioning，讓元素可綁定錨點。綁定的元素可設定為 relative 以依其錨點元件調整大小與位置。

  • 網路平台

    支援 Storage-Access-Headers，允許伺服器在取得 Storage Access API 授權後，透過 HTTP header 使用未分區 cookie。

  • 網路平台

    實作 CSS 根字型相對單位（root-font-relative units） rcap、rch、rex 與 ric。

  • 請參考此版本的完整變更清單。 你可能也想瞭解 Firefox 版本更新資訊。
