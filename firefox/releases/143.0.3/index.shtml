Firefox 新鮮事

  • 修正

    調整「拖曳以釘選」功能提示的時機，以提高可靠性。（Bug 1989344）

  • 修正

    修正部分擴充套件無法透過附加元件管理員更新的問題。（Bug 1984724）

  • 修正

    修正 Firefox View 各區塊無法如預期摺疊／展開的回退錯誤（regression）。（Bug 1989200）

  • 修正

    處理 WebExtensions 儲存問題，避免 Firefox 在安裝特定擴充套件時啟動時當機。（Bug 1989840）

  • 修正

    修正從安裝後的提示打開擴充套件的「套件設定」時，可能導致「自訂工具列」與部分快捷鍵在重新啟動前失效。（Bug 1983869）

  • 修正

    降低在封鎖 UDP 的網路環境下，載入特定網站時的長時間延遲。（Bug 1980812）

  • 修正

    多項穩定性與安全性修正。
