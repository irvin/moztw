Firefox 新鮮事

  • 修正

    修正 about:settings 中可用語言的清單多於已下載或目前支援的語言。（Bug 1994642）

  • 修正

    修正使用鍵盤開啟統一搜尋網址列選單（Unified Search dropdown）的不一致問題，選單現在能正確展開，讓使用者可使用鍵盤選擇搜尋引擎。（Bug 1979826）

  • 修正

    修正 Microsoft OneDrive Photos，「For You」頁面上策展照片集合無法載入，顯示成灰色畫面而非內容。現在集合會如預期顯示了。（Bug 1986533）

  • 修正

    修正安裝 Avast 或其他安全軟體的 Windows 使用者啟動時當機。（Bug 1992678）

  • 修正

    修正 macOS 上切換應用程式後，emoji 選擇器快捷鍵和選單項目失效。（Bug 1980815）

  • 修正

    修正 macOS 上從 Firefox 拖曳圖片到第三方應用程式（如預覽程式），可能失敗或行為異常。（Bug 1995345）

  • 修正

    修正 macOS 26 (Tahoe) 上，系統處於高負載時，可能發生的效能和影片播放問題。（Bug 1995638）

  • 修正

    修正 macOS 26 (Tahoe) 上，當書籤資料夾包含循環或自我引用時，可能導致瀏覽器當掉。（Bug 1995621）

  • 修正

    多項安全性修正。
