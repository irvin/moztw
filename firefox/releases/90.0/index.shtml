Firefox 新鮮事

  • 全新功能

    在 Windows 上, Firefox 未執行時 仍可以在背景套用更新。

  • 全新功能

    Windows 版 Firefox 新增了 about:third-party 頁面，提供第三方程式造成的相容性問題資訊。

  • 全新功能

    可以在 about:preferences#privacy 中管理純 HTTPS 模式的例外網站。

  • 全新功能

    列印的「儲存為 PDF」功能會生成可使用的超連結。

  • 全新功能

    第二版的 Firefox SmartBlock 功能 進一步改善隱私瀏覽體驗。雖然仍會阻擋第三方 Facebook 程式避免你被追蹤，但當你決定使用「透過 Facebook 登入」時，會自動即時載入。

  • 修正 修補多個安全弱點。
  • 變更

    右鍵選單「用新分頁開啟圖片」功能，現在預設在背景分頁打開圖片與媒體。進一步了解。

  • 變更

    大部分未擁有支援硬體加速 WebRender 的使用者，現在會使用軟體 WebRender。

  • 變更

    改善軟體 WebRender 的效能。

  • 變更

    現已移除 FTP 支援。

  • 企業版 在最新的 Firefox 修復多個 Bug 與加入新政策。您可在 Firefox 90 企業版版本資訊找到更多資訊。
  • 開發功能 Firefox 90 開發者資訊
  • 開發功能

    開發工具支援 Private Fields （TC39 提案 第三階段），範圍包含：object inspection、autocompletion、expression evaluation、variable tooltips 與 pretty printing (bug)。

  • 開發功能

    網路監控器面板在字體相關的 HTTP 請求的回應分頁中顯示預覽。 (bug)

  • 網路平台

    支援 Fetch Metadata Request Headers，讓網路應用程式能強化自我及對使用者的保護，抵抗多種跨來源（cross-origin）威脅。

  • 網路平台

    支援使用儲存於硬體 Token 或作業系統儲存空間中的用戶端驗證憑證。
