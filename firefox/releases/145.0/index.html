Firefox 新鮮事

  • 全新功能

    新增、編輯和刪除註解，在 PDF 中建立自己的筆記（摘要、問題、任務等）。註解側邊欄可協助你瀏覽所有註解並快速跳轉，對於長篇或大量標記的 PDF 特別實用。

  • 全新功能

    此版本引入全新世代隱私保護，奠基於我們在減少瀏覽器指紋的相關研究基礎。新防護機制將被判斷為獨一無二的用戶百分比幾乎減半，確保在隱私瀏覽或 增強型追蹤保護 設為 嚴格 時，能提供更安全、更隱密的瀏覽體驗。

  • 全新功能

    需要提醒你分頁群組中有哪些分頁嗎？將滑鼠懸停在分頁群組名稱上，即可預覽其中的分頁，無需實際點開群組。

  • 全新功能

    現在無需開啟新分頁或視窗，即可直接從側邊欄存取和管理已儲存的密碼。停留在當前頁面，更快取得登入資訊。

  • 全新功能

    「複製選擇範圍的連結」功能，讓使用者可以透過右鍵選單，複製選取文字的對應連結，與他人分享頁面的任意區段。

  • 全新功能

    在翻譯不同書寫方向的語言時，翻譯功能的使用體驗有所改進。

  • 全新功能

    桌面版新分頁現在提供新的品牌風格桌布，包含淺色和深色模式版本。

  • 全新功能

    透過新的「將應用程式開啟的分頁，開啟於目前瀏覽分頁旁」設定，你可以讓來自其他應用程式的連結，開啟在活動分頁旁邊、而非分頁列的末端。

  • 全新功能

    增強型彈跳追蹤保護（Enhanced Bounce Tracking Protection）的無狀態模式，現已在 ETP 嚴格模式 中預設啟用，可阻擋更多基於重新導向的進階追蹤技術。

  • 全新功能

    對於大多數 Windows 使用者，現有的 Firefox 桌面捷徑已更換成桌面啟動器。這是一個用來啟動已安裝的 Firefox 的小型程式。如果 Firefox 尚未安裝，則會提示使用者安裝 Firefox。對於拿到新 Windows 裝置的 Firefox 使用者，這可以提供簡易的安裝點，因桌面啟動器程式可能已透過 OneDrive 或其他雲端硬碟同步到桌面。深入瞭解。

  • 修正

    多項安全性修正。

  • 變更

    Firefox 不再支援 32 位元 Linux 系統。我們建議安裝 64 位元版本以繼續獲得更新和支援。

  • 變更

    水平分頁現在稍微更圓潤，以符合垂直分頁的外觀。按鈕和文字框（包括網址列）也已同時更新，以保持一致。

  • 變更

    當未安裝擴充套件時，點擊擴充套件按鈕現在會顯示訊息，強調擴充套件如何增強你的瀏覽體驗，並提供 Firefox 附加元件網站的連結。

  • 變更

    本地翻譯模型現在使用 Zstandard 壓縮，減少下載容量和裝置儲存空間。一如既往，所有 Firefox 翻譯都在你的本地裝置上安全且私密地進行，與其他使用雲端服務翻譯的瀏覽器不同。

  • 變更

    我們更新了預設自動化偏好設定，以更好地支援代理瀏覽（Agentic browsing），減少機器人完成任務所需的步驟，並降低機器人卡住的機會。

  • 企業版

    你可以在 Firefox 145 企業版版本資訊 中找到政策更新與企業版相關的錯誤修復資訊。

  • 開發功能

    Firefox 145 開發者資訊

  • 開發功能

    Microsoft UI Automation 分階段推出已達 100%。Microsoft UI Automation 是 Windows 新的協助工具框架。這將為 Windows 上的現有和未來的協助技術，提供更好的支援和效能。

  • 網路平台

    新增對 Atomics.waitAsync 提案的支援，這是 Atomics.wait 的非阻擋、非同步版本。這允許基於共享記憶體位置中的值進行執行緒同步，更多詳細資訊請參閱 MDN 文件。

  • 網路平台

    Firefox 現在支援新的Integrity-Policy 檔頭，用於強制執行腳本的子資源完整性。此檔頭的進一步實作正在進行中。

  • 網路平台

    新增常用編碼器的 Matroska 支援：AVC、HEVC、VP8、VP9、AV1、AAC、Opus 和 Vorbis。

  • 網路平台

    現在支援 text-autospace 屬性，允許在不同書寫系統的字元之間自動調整間距。

  • 網路平台

    WebGPU DOM API（規格／MDN）在 Apple Silicon 的 macOS 26 (Tahoe) 上現已可用。詳細資訊請參閱 Mozilla Graphics Team Blog。

下載與安裝

系統需求

在安裝 Firefox 前請先確定您的電腦是否合乎系統需求。

下載 Firefox

您可以由此下載最新的 Firefox，或取得 Windows、Linux 及 Mac OS X 平台上的各國語言版本，或由此下載 Firefox 未來發行版本（測試版）。若需自行編譯其他平台或語言的版本，請見本文件最末節。

安裝 Firefox

請注意：安裝新版 Firefox 將覆蓋現存的 Firefox 程式，您不會遺失書籤或瀏覽記錄，但有些附加元件可能要更新才能繼續使用。詳細安裝步驟請參見如何安裝與更新 Firefox。

移除 Firefox

在 Windows 作業系統中，您可以使用「新增或移除程式」移除 Firefox。macOS 的使用者請將 Firefox 程式移至垃圾筒；Linux 使用者則請移除包含 Firefox 的資料夾。

一般狀況下，移除 Firefox 時不會把書籤、瀏覽記錄、擴充套件和其他附加元件一併刪除，這些資料會保留在您的「個人設定資料夾」內，此資料夾可以在「說明」選單的「更多疑難排解資訊」找到，按下「應用程式一般資訊」中「設定檔目錄」旁的按鈕便可開啟它。

請留意，若您沒有手動刪除個人設定資料夾，移除後重新安裝 Firefox 將繼續沿用原本的設定檔。

附加元件及佈景主題

您可以由「附加元件管理員」或附加元件網站取得附加元件及佈景主題。 當 Firefox 升級時，附加元件可能因不相容於新版 Firefox 而被停用，此時則需要原作者進行升級。

疑難排解

  • 設計不良或不相容的套件將導致瀏覽器發生問題，包含當機或網頁瀏覽速度變慢等等狀況。如果您碰上瀏覽器無法開啟、視窗顯示不正常等等狀況，便可能是擴充套件或佈景主題的問題。這時請參考「Firefox 輔助說明：安全模式」一文開啟 Firefox 安全模式。在安全模式下，所有附加元件及佈景主題將被停用。請您在安全模式下移除有問題的擴充套件或佈景主題，再正常地啟動瀏覽器。
  • 如果您遇到與書籤、下載、視窗位置、工具列、瀏覽記錄或其他奇怪問題，建議您在回報問題前先建立新的使用者設定檔試試。請參考「Firefox 輔助說明：使用設定檔管理員建立或移除 Firefox 設定檔」一文建立新的使用者設定檔，並將各種設定（書籤、儲存的密碼等）從舊的設定檔中分次轉移到新的設定檔（可透過 Firefox Sync 同步功能進行）。

常見問題集

  1. 我該怎麼幫忙呢？

    我們需要開發員和測試社群回饋建議，幫我們將 Firefox 變得更盡善盡美。請先查閱 Firefox 輔助說明看看有沒有相關解答，再到 Bugzilla 回報你發現的錯誤；您也可以使用意見回饋功能表達意見。

  2. 我如何預覽未來的 Firefox 版本？

    加入 Aurora 或 Beta 開發頻道就可以搶先試用 Firefox 開發中版本。

  3. 我該到哪裡才能用中文問問題跟表示意見？

    請使用 Firefox 討論區，MozTW 社群團隊與熱心的 Firefox 用戶會定期閱讀上面的文章。

  4. 我該去哪裡取得新的佈景主題與擴充功能 (附加元件)？

    請到 Mozilla 附加元件中心下載 Firefox 的擴充套件及佈景主題。

  5. Firefox 是哪些人做出來的？

    Firefox 是由世界各地的 Mozilla 社群成員與員工合力開發。這裡有份長長的名單，列出部分曾對 Firefox 有重要貢獻的人。想要加入我們的大家庭嗎？或者加入 Mozilla 台灣社群？

  6. Firefox 的原始碼在何處？

    請參照 Getting Mozilla Source Code Using Mercurial 一文取得 Firefox 原始碼；如需編譯 Firefox，請參考 Build Instructions。

  7. 電子郵件程式呢？

    Firefox 能與您預設的任何郵件程式協同運作，而我們推薦您試試我們下一代的郵件軟體：與 Firefox 合作無間的 Mozilla Thunderbird。
