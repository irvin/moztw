Firefox 新鮮事

  • 全新功能

    設定頁新增 AI Controls 區塊，可管理 AI 強化功能。可參考 說明文章 與 操作教學。

  • 全新功能

    Firefox 強化了 PDF 中數學公式對螢幕閱讀器的支援，提升可及性體驗。

  • 全新功能

    Remote improvements 現在與遙測設定脫鉤。即使你關閉遙測與實驗研究，也可以選擇接收遠端瀏覽器改進。

  • 全新功能

    Firefox Backup 現在在 Windows 10 也支援「關閉 Firefox 時清除瀏覽記錄」的使用者。設為關閉時會清除的資料，不會被納入備份。

  • 全新功能

    翻譯功能新增語言：繁體中文雙向翻譯，以及越南語翻譯。

  • 全新功能

    New Tab 桌布現在也會套用到容器分頁的新分頁頁面。

  • 修正

    修正在重大更新後語言套件可能被停用，導致介面顯示錯誤語言的問題。

  • 修正

    在 Windows 上，將下載後的圖片拖曳到 Adobe Illustrator 時，現在會正確插入圖片本體，而不是網址。

  • 修正

    多項安全性修正。

  • 企業版

    你可以在 Firefox 148 企業版版本資訊 中找到政策更新與企業版相關的錯誤修復資訊。

  • 開發功能

    Firefox 148 開發者資訊

  • 網路平台

    初始 about:blank 文件現在採用更符合 Web 相容性的同步導覽行為。

  • 網路平台

    新增 Service Worker 對 WebGPU 的支援，WebGPU 現可在所有 worker context 使用。

  • 網路平台

    支援 Iterator.zip() 與 Iterator.zipKeyed()。

  • 網路平台

    支援 Trusted Types API，加強跨站腳本防護能力。

  • 網路平台

    支援 Sanitizer API，並新增 element.setHTML() 與 document.parseHTML()。

  • 網路平台

    支援 location.ancestorOrigins 屬性。

  • 網路平台

    Navigation API 新增 NavigationPrecommitController.addHandler() 支援。

  • 網路平台

    CSS Anchor Positioning 新增 position-try-order 支援。

  • 網路平台

    CSS 新增 shape() 函式支援，可在 clip-path 等屬性中定義可回應式自由形狀。

  • 請參考此版本的完整變更清單。 你可能也想瞭解 版本的更新。
