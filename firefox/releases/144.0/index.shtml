Firefox 新鮮事

  • 全新功能

    在分頁群組中專注單一分頁，避免雜亂。你的活動分頁現在會在分頁列中保持可見狀態，即使群組收合時也能保持整齊。

  • 全新功能

    我們很高興的分享另一個社群最期待的分頁群組新功能！現在你可以將分頁拖曳到已收合的群組中，不會因此自動展開群組。這樣能快速整理乾淨，同時最小化視覺干擾。

  • 全新功能

    設定檔管理功能將在未來幾週逐步向全球使用者推出。透過將你的線上生活分離成不同的設定檔，這個功能將幫助你保護隱私並保持專注，例如區分工作、學校、假期規劃或其他用途。你可以為設定檔命名、自訂圖示和色彩主題，以便輕鬆識別。快速在設定檔之間切換，以完全隔離書籤、分頁和瀏覽歷史。設定檔管理功能適用於 Windows 11、Mac 和 Linux 使用者，Windows 10 支援將於稍後推出。

    此功能逐步釋出中：某些新的 Firefox 功能採行逐步釋出（progressive roll out），意味著某些使用者會比其他人更早看到此功能。如此有助於獲取早期回饋，迅速發現錯誤與改善行為，從而提升整體 Firefox 使用者的體驗。

  • 全新功能

    現在可以在不暫停影片的情況下關閉子母畫面。按住Shift + 點擊 關閉按鈕，或使用 Shift + Esc 退出子母畫面，同時保持播放狀態。

  • 全新功能

    儲存在 Firefox 密碼管理員中的登入資訊，現在以更現代化的加密方案（AES-256-CBC）加密儲存於磁碟中，取代較舊的 3DES-CBC。此更新改善本地資料保護。透過 Firefox 同步的登入資訊，則早已使用 AES-256-GCM，並維持端對端加密。

  • 全新功能

    由 Google 智慧鏡頭支援的視覺搜尋

    在任何圖片上點擊右鍵，你就可以：
    ✨ 尋找類似產品、地點或物件
    ✨ 複製、翻譯或搜尋圖片中的文字
    ✨ 獲得學習、旅行或購物的靈感

    在你的右鍵選單中尋找新的「使用 Google Lens 搜尋圖片」選項（會先以 NEW 徽章標示）。

    這是一個在全球範圍推出，僅限桌面版的功能。你必須預設搜尋引擎為 Google 以使用。

    此功能逐步釋出中：某些新的 Firefox 功能採行逐步釋出（progressive roll out），意味著某些使用者會比其他人更早看到此功能。如此有助於獲取早期回饋，迅速發現錯誤與改善行為，從而提升整體 Firefox 使用者的體驗。

  • 全新功能

    內建於 Firefox 中的 Perplexity AI 搜尋

    Firefox 桌面版現在內建 Perplexity，瀏覽器中的 AI 驅動問答引擎。Perplexity 為複雜問題提供直接、對話式的回答，幫助你進行快速摘要、尋找精準的參考資料或創意靈感，無需挖掘多重來源。你可從網址列透過統一搜尋按鈕使用，此功能在全球範圍推出。

    此功能逐步釋出中：某些新的 Firefox 功能採行逐步釋出（progressive roll out），意味著某些使用者會比其他人更早看到此功能。如此有助於獲取早期回饋，迅速發現錯誤與改善行為，從而提升整體 Firefox 使用者的體驗。

  • 開發功能

    Firefox 144 開發者資訊

  • 開發功能

    現在可以從樣式規則中的 var() 函數跳轉到 CSS 自訂屬性（CSS custom property）的定義。

  • 開發功能

    檢測器中的事件工具提示（event tooltip）現在會在自訂事件旁邊顯示徽章，讓它們更容易與內建事件區分。

  • 網路平台

    Firefox 現在支援 Element.moveBefore API。

  • 網路平台

    Firefox 現在支援 math-shift compact。

  • 網路平台

    Firefox 現在支援 PerformanceEventTiming.interactionId，允許開發者將相關的輸入事件分組。此功能提供互動到下一個繪製（INP）回應指標支援。

  • 網路平台

    Firefox 現在支援 command 和 commandfor 屬性。

  • 網路平台

    Firefox 現在支援 View Transitions API Level 1。View Transitions API 提供機制輕鬆在不同網站視圖（views）間建立轉換動畫。

  • 網路平台

    當使用 WebRender 硬體渲染 linear-gradient、conic-gradient 和 radial-gradient 時，現在會套用抖色（dithering）。

  • 網路平台

    Firefox 現在支援 upsert 提案。這為 Map 和 WeakMap 新增了 getOrInsert 和 getOrInsertComputed 方法，可直接回傳與鍵關聯的值，或者先插入後回傳預設值，簡化不知道特定 Key 是否已存在 Map 或 WeakMap 的操作。

  • 網路平台

    Firefox 現在在 Windows 平板和 Android 裝置上支援 ScreenOrientation 介面的 lock() 和 unlock() 方法。

  • 網路平台

    Firefox 現在支援 RTCDataChannel 以 worker transfer 傳遞。

  • 網路平台

    Firefox 現在支援 resizeMode 的 getUserMedia 對象，讓開發者可裁剪和縮放攝影機擷取的影片到指定的任何解析度。

  • 網路平台

    Firefox 現在在 Windows 上支援 WebGPU 的 GPUDevice.importExternalTexture API。

  • 網路平台

    在 Windows 上 WebCodecs 現在為 VideoEncoder 提供批次編碼路徑。由於預設批次大小更大，可改善效能、提供更高的吞吐量，以及更低的提交延遲。

  • 網路平台

    Gecko 專有的 CSS2Properties 改名為 CSSStyleProperties，以符合最新的網頁標準，並與其他瀏覽器引擎有更好的互操作性。

  • 請參考此版本的完整變更清單。 你可能也想瞭解前一個版本的更新。
