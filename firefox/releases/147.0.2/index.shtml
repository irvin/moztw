Firefox 新鮮事

  • 全新功能

    Firefox 現在提供實驗性的快捷鍵自訂功能。你可以在網址列輸入 about:keyboard，替換難按或難記的組合鍵、避開與其他軟體衝突，打造自己慣用的快捷鍵配置。歡迎到 Mozilla Connect 分享回饋。

  • 修正

    修正 Linux 使用 XDG Base Directories 時，瀏覽器部分功能缺失或失效的多項問題。（Bug 2011300）

  • 修正

    修正登入部分網站時會出現過多 passkey 提示視窗的問題。（Bug 2010919）

  • 修正

    修正網站被 SafeBrowsing 錯誤標記為惡意站台的問題。（Bug 2010956）

  • 修正

    多項安全性修正。
