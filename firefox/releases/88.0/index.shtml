Firefox 新鮮事

  • 全新功能

    PDF 表單現在支援 嵌入於 PDF 檔案內的 JavaScript。有一些 PDF 表單使用 JavaScript 來進行驗證，或提供其他互動功能。

  • 全新功能

    列印功能更新：文件邊界（Margin）的單位目前已ㄎ翻譯。

  • 全新功能

    Linux 開始支援觸控板進行畫面平滑縮放（pinch-zooming）。

  • 全新功能

    為了避免跨網站隱私外洩，Firefox 現在會隔離 window.name 的資料，僅限建立的網站可取用。

  • 修正

    螢幕閱讀器現在會避免讀出網站內視覺上隱藏的內容，如 Google Help 面板中的文章。

  • 修正 數個安全性修正。
  • 變更

    如果你在過去 50 秒內曾在相同網站的同一個分頁內許可使用攝影機與麥克風的，Firefox 會避免再次要求你許可存取。這個新的寬限期，將減少收到裝置權限請求的次數。

  • 變更

    畫面截圖功能將從網址列中的網頁動作選單移除。請透過右鍵選單拍攝螢幕截圖。你也可以透過自訂選單功能，新增「畫面截圖」按鈕到工具列上：打開 Firefox 選單並選擇「自訂...」。

  • 變更

    FTP 支援已預設停用，並且將在未來的版本中完全移除。此變更是為了減少相關的安全問題，並漸次停止支援未經加密的通訊協定。

  • 企業版 在最新的 Firefox 修復多個 Bug 與加入新政策。您可在 Firefox 88 企業版版本資訊找到更多資訊。
  • 開發功能 Firefox 88 開發者資訊
  • 開發功能

    「網路」面板中新增「原始」切換鈕，可切換以 JSON 格式顯示的 HTTP 回應或原始資料（一如從線上直接收到的資訊）。

  • 網路平台

    開始支援 Event Timing API。

  • 網路平台

    開始支援 CSS forced-colors media query。
