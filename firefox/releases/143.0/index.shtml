Firefox 新鮮事

  • 全新功能

    在 Windows 上，Firefox 現在支援將網站作為網頁應用程式（Web App），直接釘選到工作列並以精簡視窗執行，同時也能保留已安裝的附加元件的可用性。（目前不適用於從 Microsoft Store 安裝的 Firefox）

  • 全新功能

    現在可以將分頁拖曳到分頁列的最前端來釘選，使常用網站觸手可及。

  • 全新功能

    現在可選擇 Microsoft Copilot 作為 側邊欄聊天機器人，便於在不中斷主要工作的情況下快速取用 AI。

  • 全新功能

    當網站要求攝影機權限時，權限對話框內可直接預覽攝影機畫面，在多台攝影機間切換時特別方便。

  • 全新功能

    Firefox 網址列現在可顯示美國、英國、德國、法國與義大利地區的重要日期與活動（如「母親節」）。

  • 全新功能

    Firefox 現在會在 更多裝置屬性 上回報固定值，以擴大數位指紋追蹤防護（Fingerprinting Protection）。

  • 全新功能

    在隱私視窗下載檔案後，當結束工作階段時，會詢問是否要保留或刪除該檔案；你也可以在設定中調整此預設行為。

  • 全新功能

    Firefox 現在支援 Windows UI Automation，可改善 Windows 語音存取、文字游標與朗讀程式等協助工具的相容性。

    此功能逐步釋出中：某些新的 Firefox 功能採行逐步釋出（progressive roll out），意味著某些使用者會比其他人更早看到此功能。如此有助於獲取早期回饋，迅速發現錯誤與改善行為，從而提升整體 Firefox 使用者的體驗。

  • 修正

    多項安全性修正。

  • 企業版

    你可以在 Firefox 143 企業版版本資訊 中找到政策更新與企業版相關的錯誤修復資訊。

  • 開發功能

    Firefox 143 開發者資訊

  • 開發功能

    取消勾選「將類似訊息組成一組」後，連續的相似訊息將不再被群組，所有訊息都會在輸出處顯示。

  • 開發功能

    在除錯器中切換未修改與「Pretty Print 原始碼」時，不再開啟新分頁顯示 Pretty 內容。

  • 開發功能

    當檢測器聚焦（focus）時，現在可透過 Ctrl+Shift+Y 快捷鍵啟動滴管工具，以挑選畫面上的顏色。

  • 網路平台

    Firefox 現在在 Windows 11 22H2+、macOS 與 Android 9+ 上支援 xHE-AAC 音訊播放。

  • 網路平台

    Firefox 更新了 CSS Grid 尺寸演算法，使之更貼近標準。使用百分比列尺寸或包含具長寬比元素（如圖片）的格線項目之版面在更多情況下能正確呈現。

  • 網路平台

    <input type=color> 現在除了十六進位色碼（#ffffff）外，也能辨識 CSS 的 <color> 格式，如顏色名稱（例如 black）與更複雜的表示法（如 rgb(200 200 200)）。目前輸入值仍會轉換為十六進位格式。

  • 網路平台

    已移除限制，現在可設定 <details> 的 display 屬性，並新增 ::details-content 偽元素，以便為元件的展開／收合內容設定樣式。

  • 請參考此版本的完整變更清單。 你可能也想瞭解前一個版本的更新。
