Firefox 新鮮事

  • 在 Mozilla，我們相信您有隱私權。您不該在上線時被追蹤，不論您正在查看銀行餘額、尋找最好的醫師或者買鞋子，不道德的追蹤公司不該有辦法在您瀏覽網路時追蹤您。因此，我們持續強化 Firefox，以替我們的使用者抵抗線上追蹤。
  • 全新功能 在隱私視窗及加強型追蹤保護的「嚴格」模式下，SmartBlock 會減少您遇到網站破損的情況。該功能提供了替代腳本，使網站可以正確載入。
  • 全新功能 為了進一步保護您的隱私，我們新的預設 HTTP Referrer 政策將去除 Referrer 標頭的路徑（path）和查詢字符串（query string）資訊，以防止站點意外洩漏敏感的用戶數據。
  • 全新功能 現在，「在頁面中搜尋」上的「強調全部」功能，會在視窗捲軸上顯示刻度線，對應頁面上匹配項目的位置。
  • 全新功能 我們很自豪地宣布全面支持 macOS 內置螢幕閱讀器 VoiceOver。
  • 全新功能 我們添加了一個新的語系：西里西亞語（szl）
  • 修正 我們修復了幾個重大的輔助功能（a11y）問題：
    • 影像控制元件現在具有可見的聚焦（focus）樣式，並且可以透過鍵盤操控影像和聲音控制元件。 (Bug 1681007)
    • 現在螢幕閱讀器會唸出 HTML <meter> 元件。 (Bug 1460378)
    • Firefox 現在在「附加元件管理員」中設置了實用的初始焦點。 (Bug 580537)
    • 現在，當 aria-labelledby/describedby 內容改變時，Firefox 將觸發一個 name/description 變動（change）事件。 (Bug 493683)
  • 修正 修補幾種不同的安全弱點。
  • 變更 為了防止用戶在填寫表單時丟失數據，我們已停用 Backspace 鍵做為後退按鈕的導覽捷徑。要重新啟用 Backspace 鍵盤快捷鍵，可以將 about:config 中的 browser.backspace_action 更改為 0。您也可以改用建議的 Alt + 左​​箭頭（在 Mac 上為 Command + 左箭頭）
    Firefox 鍵盤快捷鍵
  • 變更 我們已從「收藏庫」按鈕選單中刪除以下不常用、或瀏覽器中有其他存取點的項目：「同步的分頁」，「近期焦點」和「檢視 Pocket 清單」。
  • 變更 我們簡化了「說明」選單，移除重複項目，諸如可透過「取得幫助」獲取的那些指向 Firefox 技術支援頁面的內容。
  • 企業版 多個 Bug 修復與新政策加入到最新的 Firefox 版本。您可在 Firefox 87 企業版版本資訊找到更多資訊。
  • 開發功能 Firefox 87 for developers 開發者資訊
  • 開發功能 我們大幅簡化了「網頁開發者」選單。請改由應用程式選單（漢堡鈕）> 網頁開發者 > 網頁開發者工具 來使用檢測器、主控台、除錯器、網路、樣式編輯器、效能、記憶體、儲存空間、輔助功能、應用程式等工具。
  • 開發功能 開發者現在可以使用檢測器模擬 prefers-color-scheme media queries，而不必更改作業系統的亮或暗模式。
  • 開發功能 除了以前支援的 :hover、:active、:focus、:focus-within、:focus-visible 和 :visited 這些 pseudo-classes 以外，開發者現在可以使用檢測器為當前選定的元素切換 :target pseudo-class
  • 開發功能 與未生效的 CSS 規則有關的檢測器改良及錯誤修復：
    • table-layout 屬性對於 table 以外的元素會被標記為無用。
    • scroll-padding 屬性（縮寫和全稱）對於無法捲動的元素會被標記為無用。
    • text-overflow 屬性之前在一些 overflow 值會錯誤的顯示為無用。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
